UNPRFCT

Unprfct is an online shopping portal for fashion and lifestyle products, where customer can buy products and admin can add/delete products.


Technologies Used
  • Angular 8
  • HTML 5
  • CSS 3
  • Bootstrap 5
  • MySQL
  • SpringBoot 2.5
  • JPA
  • Rest API
Notable Features
  • Login/Register
  • Can add/delete multiple products to cart.
  • Search a product or filter based on price,brand and color.
  • Can view and track orders.
My Role
  • In this project I oversaw the entire SDLC.
Project Difficulties
  • Cart Funtionality.
  • Persist the cart data for a long time until the user buy it.
